# Astra-Font-Oficial
Descripcion: Astra-Style-Rg es una tipografia experimental con estetica futurista y moderna, diseñada especificamente para el universo de Astra Corp. Diseñada con imperfecciones que buscan generar una pequeña desconformidad al espectador sin saturarlo.

Version: 004.002

Estilo: Regular Stencil

Metricas Personalizadas:
Letras Con Espacios: 60
Letras Redondas: 61
Simbolos: 62
Letras Rectas: 63

Compatibilidad: Diseñada para Windows y Android.

Formato: TrueTypeFont (ttf)

Detalles del Proyecto: Esta fuente nace como parte de una historia (actualmente en desarrollo, sin titulo definitivo) ambientada en una gelida isla y un mundo controlado por una corporacion mysteriosa llamada Astra Corp.
Los glifos basicos comenzaron hechos en digital y pasados por Calligraphr, luego se pasaron por FontForge para ajustar medidas, margenes y agregar más glifos, ademas, se ajustaron ciertos glifos. (glifos con imperfecciones hechos adrede como detalle tecnico para representar a la compania en la historia intentando generar cierta desconformidad leve al espectador)
